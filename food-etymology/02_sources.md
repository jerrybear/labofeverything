# 샤브샤브 어원 연구 출처

## 문서 목적

- 이 문서는 `food-etymology/01_shabu-shabu-etymology.md`의 핵심 근거 출처를 정리한다.
- 우선순위는 `공식 설명/사전류 -> 보조 해설` 순서다.

## 1. 최우선 출처

### 1.1 코토뱅크 `しゃぶしゃぶ`

- URL: `https://kotobank.jp/word/しゃぶしゃぶ-3154275`
- 성격: `사전 종합 포털`
- 이번 연구에서 활용한 포인트:
  - `しゃぶしゃぶ`가 본래 `가볍게 물을 휘저어 씻거나 헹굴 때의 소리/모습`이라는 설명
  - 음식 뜻으로는 `중국 요리 涮羊肉 등에서 나온 요리`라는 설명
  - 비교적 이른 용례와 사전적 정의 확인

### 1.2 堂島スエヒロ `しゃぶしゃぶのネーミング`

- URL: `https://www.e-suehiro.com/dojima/history/naming/`
- 성격: `브랜드 공식 회고`
- 이번 연구에서 활용한 포인트:
  - 물수건을 헹구는 소리를 듣고 이름을 붙였다는 설명
  - `しゃぶしゃぶは肉の洗濯や`라는 브랜드 서사

### 1.3 永楽町スエヒロ本店 `しゃぶしゃぶ60周年`

- URL: `https://www.e-suehiro.com/syabu60th/`
- 성격: `브랜드 공식 역사 정리`
- 이번 연구에서 활용한 포인트:
  - `1952년 개발`, `1958년 상표 등록` 설명
  - 스에히로가 음식명 대중화의 핵심 축이었다는 점

## 2. 보조 출처

### 2.1 しゃぶ葉 `しゃぶしゃぶの起源とは？`

- URL: `https://www.skylark.co.jp/syabuyo/syabusyabu/index.html`
- 성격: `외식 브랜드 해설 페이지`
- 이번 연구에서 활용한 포인트:
  - 베이징의 `涮羊肉`과 전후 일본식 변형 서사 정리
  - 대중적으로 유통되는 기원 설명의 형태 확인

### 2.2 語源由来辞典 `しゃぶしゃぶ`

- URL: `https://gogen-yurai.jp/syabusyabu/`
- 성격: `보조 어원 해설`
- 이번 연구에서 활용한 포인트:
  - 스에히로, 삼택충일, 상표 등록 관련 대중적 정리
  - 브랜드 회고와 사전류 설명이 대중 해설에 어떻게 요약되는지 확인

## 3. 읽을 때의 주의점

- 스에히로 자료는 `1차 회고 자료`에 가깝지만 동시에 `브랜드 서사`이기도 하다.
- 외식 브랜드나 어원 해설 사이트는 이해하기 쉽지만, 세부 연도와 직접 전승 경로는 교차 검증이 필요하다.
- 가장 안전한 결론은 `의성·의태어적 명칭 + 전후 일본 상품화 + 중국식 선행 조리법 영향`까지다.
