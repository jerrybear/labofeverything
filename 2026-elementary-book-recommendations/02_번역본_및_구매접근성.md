# 해외 추천작 번역본 및 구매 접근성 메모

## 확인 기준

- 2026-03 기준 공개 웹 검색과 국내 주요 서점 검색 결과를 기준으로 정리했다.
- 여기서 `번역본 확인`은 국내 번역본 정보를 이번 조사 범위에서 직접 확인한 경우를 뜻한다.
- `외서 구매 가능`은 교보문고 또는 알라딘 검색 결과에서 외국도서로 확인된 경우를 뜻한다.

## 정리표

| 도서 | 국내 번역본 확인 | 외서 구매 가능 | 메모 |
| --- | --- | --- | --- |
| `Fireworks` | 확인 못함 | 가능 | 교보문고 검색 결과에서 외서 1종 확인 |
| `Stop That Mop!` | 확인 못함 | 가능 | 알라딘 검색 결과에서 외국도서로 확인 |
| `All the Blues in the Sky` | 확인 못함 | 가능 | 교보문고, 알라딘에서 외서 중심 노출 |
| `The Nine Moons of Han Yu and Luli` | 확인 못함 | 가능 | 알라딘에서 외국도서 하드커버 확인 |
| `A Sea of Lemon Trees` | 확인 못함 | 재확인 필요 | 이번 조사 범위에서 국내 서점 검색 재확인 미진행 |
| `The Teacher of Nomad Land` | 확인 못함 | 재확인 필요 | 수상 정보는 확인, 국내 판매 여부는 후속 확인 필요 |
| `The Undead Fox of Deadwood Forest` | 확인 못함 | 재확인 필요 | 수상 정보는 확인, 국내 판매 여부는 후속 확인 필요 |

## 해석

- 2026 수상 해외 아동서는 국내 서점에서 우선 외서 형태로 먼저 유통되는 경우가 많다.
- 이번 조사에서 바로 한국어 번역본을 확인한 작품은 없었다.
- 실제 수업 적용이나 선물용 목록으로 쓸 때는 번역본 출간 시차를 꼭 감안해야 한다.

## 실사용 팁

### 번역본이 바로 필요한 경우
- 해외 수상작은 "후보"로 두고, 실제 목록에는 국내 출간작을 우선 배치하는 편이 안전하다.
- 해외 수상작을 넣더라도 `번역본 확인 후 추가`로 표시해 두는 것이 좋다.

### 영어 원서 활용이 가능한 경우
- 국제학교, 영어도서관, 원서 읽기 모임에서는 `Fireworks`, `Stop That Mop!`, `All the Blues in the Sky` 같은 책을 바로 활용할 수 있다.
- 저학년은 그림책과 리더스, 고학년은 뉴베리 계열 작품부터 시작하는 편이 무난하다.
