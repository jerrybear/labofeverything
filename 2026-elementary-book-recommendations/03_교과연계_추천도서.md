# 교과 연계 추천도서 정리

## 태그 기준

- `국어`: 읽기, 말하기, 쓰기, 문학 감상
- `사회`: 역사, 공동체, 시민성, 문화 이해
- `과학`: 자연 관찰, 생태, 탐구, 과학 개념
- `도덕`: 감정, 관계, 공감, 책임, 평등
- `예술`: 그림 읽기, 음악, 표현, 감각 경험
- `수학`: 수 개념, 논리, 문제 해결

| 도서 | 권장 학년 | 교과 연결 포인트 |
| --- | --- | --- |
| `Fireworks` | 1-2학년 | 그림을 보고 말하기, 장면 묘사, 감각 언어 |
| `신기한 시간표` | 3-4학년 | 단편 구조 이해, 상상 이어쓰기 |
| `All the Blues in the Sky` | 5-6학년 | 운문과 산문 비교, 감정 표현 글쓰기 |
| `친구 자판기` | 1-2학년 | 인물 마음 읽기, 대화 이어 말하기 |
| `하고 싶은 공부` | 5-6학년 | 자기 주제 글쓰기, 진로 말하기 |

## 바로 쓰기 좋은 태그 예시

- `Fireworks` - `[국어] [예술] [1-2학년]`
- `왜왜왜 동아리` - `[사회] [도덕] [과학] [4-6학년]`
- `합리적인 의심` - `[국어] [도덕] [사회] [5-6학년]`
- `세상을 담는 저울` - `[사회] [도덕] [3-6학년]`
- `누가 누구?` - `[과학] [3-4학년]`

## 추천 운영 방식

- 학급문고는 `국어 2권 + 사회 2권 + 과학 2권 + 도덕 2권`처럼 섞는 편이 좋다.
- 독서수업은 한 권을 여러 교과에 걸쳐 읽히는 방식이 효율적이다.
- 5-6학년은 사회·도덕·국어를 묶어 토론형 독서로 확장하기 좋다.
