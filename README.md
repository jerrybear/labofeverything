# 리서치 워크스페이스

이 폴더는 전체 리서치 작업을 관리하는 루트입니다.

본 문서와 하위 항목 설명은 100% AI 생성 내용입니다.

## 용도

- 주제별 조사 내용을 하위 폴더 단위로 정리합니다.
- 재사용 가능한 노트와 생성 문서는 Markdown으로 관리합니다.
- 루트에는 전체 워크스페이스 안내와 공통 맥락만 둡니다.

## 현재 주제 폴더

아래 목록은 현재 작업 트리가 아니라, 현재 `git`에 커밋된 루트 하위 폴더만 기준으로 정리했습니다.

- `2026-elementary-book-recommendations/`: 초등학생 추천도서 목록, 선정 기준, 교과 연계, 학부모·학교 배포용 자료를 함께 정리한 폴더입니다.
- `TFT/`: 팀파이트 택틱스 최신 패치 기준 메타 가이드, 체크리스트, 초반 운영 판단 자료를 정리한 폴더입니다.
- `ai-orchestration/`: AI 오케스트레이션 개념, 프레임워크 비교, 운영 패턴을 정리한 문서 묶음입니다.
- `ai-platform-planner-workflow/`: AI 시대 플랫폼 기획자의 업무 방식, 의사결정 기준, 운영 템플릿을 연구한 폴더입니다.
- `balance/`: 용어 정의, 결제 주기, 자금 흐름, 엣지 케이스, 체결 엔진까지 결제 구조를 체계화한 폴더입니다.
- `career-transition/`: 개발자 은퇴 이후 진로, 한국 시장 신호, 자문·강의·마이크로 SaaS 가능성을 검토한 폴더입니다.
- `mcu-ai-agents/`: MCU 인물을 통해 AI 에이전트의 판단, 책임, 신뢰 문제를 탐구하는 창작형 연구 폴더입니다.
- `nasdaq-stock-token-trading/`: 주식 토큰화와 온체인 증권시장 가설, 거래 구조, 관련 소스를 정리한 폴더입니다.
- `parenting-boy/`: 연령별 체크리스트, 대화 예시, 부모 역할, 문제 상황 대응을 포함한 남아 양육 가이드 폴더입니다.
- `parenting-girl/`: 연령별 체크리스트, 대화 예시, 부모 핸드북 중심의 여아 양육 가이드 폴더입니다.
- `physics-after-newton/`: 뉴턴 이후 물리학의 사상 전개와 Agentic AI 시대를 연결해 보는 연구 문서 폴더입니다.
- `us-private-credit-instability/`: 미국 사모대출 시장의 불안 신호와 구조적 취약성을 다루는 조사 폴더입니다.
- `너에게-조금-느리게-가고-있어/`: 장편 서사 초안, 회차별 구조 메모, 등장인물과 세계관 설정을 쌓아 가는 창작 폴더입니다.
- `요구사항은 크루아상으로 주세요/`: 요구사항 분석을 크루아상 비유로 풀어가는 장편 초안, 편집 메모, 통합 원고를 담은 창작 폴더입니다.
- `인물-중심-서양음악사/`: 인물, 도시, 장르, 영향 관계를 중심으로 서양음악사를 재구성하는 연구 폴더입니다.
- `인터넷-지불결제-역사/`: 인터넷 환경과 한국 인터넷 결제의 역사 변화를 정리한 폴더입니다.
- `종이-역사-현재-미래/`: 종이의 역사적 발전, 현재 용도, 미래 전망, 한국 제지산업과 비교 자료를 모은 폴더입니다.
- `한국_근현대사_인물관계사/`: 시기 구분, 인물카드, 관계카드, 출처 메모로 한국 근현대사 인물 관계를 추적하는 연구 폴더입니다.

## 운영 메모

- 주제별 상세 문서는 각 하위 폴더 안에서 관리합니다.
- 루트 파일은 최소한으로 유지하고, 탐색과 안내 중심으로 사용합니다.
