# 리서치 워크스페이스

이 폴더는 전체 리서치 작업을 관리하는 루트입니다.

이 README를 포함해 이 아래 작성된 내용은 100% AI가 작성한 문서입니다.

## 용도

- 주제별 조사 내용을 하위 폴더 단위로 정리합니다.
- 재사용 가능한 노트와 생성 문서는 Markdown으로 관리합니다.
- 루트에는 전체 워크스페이스 안내와 공통 맥락만 둡니다.

## 현재 주제 폴더

- `2026-elementary-book-recommendations/`: 2026년 기준 초등학생 추천도서와 학부모·학교 배포용 파생 문서를 정리한 폴더입니다.
- `ai-orchestration/`: AI 오케스트레이션 개념, 프레임워크 비교, 운영 패턴을 학습 순서에 맞춰 정리한 문서 묶음입니다.
- `ai-platform-planner-workflow/`: AI 시대 플랫폼 기획자의 업무 방식, 의사결정 기준, 운영 템플릿을 연구한 폴더입니다.
- `mcu-ai-agents/`: MCU 인물들을 통해 AI 에이전트의 판단, 책임, 신뢰, 인간성 문제를 탐구하는 창작형 연구 폴더입니다.
- `physics-after-newton/`: 뉴턴 이후 물리학의 사상 전개와 Agentic AI 시대를 연결해 보는 대담형 연구 문서 폴더입니다.
- `us-private-credit-instability/`: 미국 사모대출 시장의 불안 신호, 구조적 취약성, 투자자 관점 체크리스트를 다루는 조사 폴더입니다.
- `인물-중심-서양음악사/`: 서양음악사를 인물 중심으로 재구성하기 위한 자료, 링크 후보, 관계도 메모를 모은 폴더입니다.

## 운영 메모

- 주제별 상세 문서는 각 하위 폴더 안에서 관리합니다.
- 루트 파일은 최소한으로 유지하고, 탐색과 안내 중심으로 사용합니다.
