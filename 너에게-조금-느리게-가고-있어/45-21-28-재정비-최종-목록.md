# 21~28 재정비 최종 목록

이 문서는 21화~28화 구간의 재정비 이후 최종 사용 흐름을 정리한다.

## 결론

이 구간은 기존 8화 체제보다, 재정비 결과를 반영한 `5블록 체제`로 읽는 편이 가장 자연스럽다.

현재 최종 권장 흐름은 다음과 같다.

1. `F-21-원고.md`
2. `F-22-원고.md`
3. `F-23-원고.md`
4. `F-24-원고.md`
5. `F-25-원고.md`

## Canon 블록 목록

### 1. `F-21-원고.md`

- 역할: 병원 내부 블록 종결
- 기능: 되밀림 -> 피해자 실재 -> 이름 덧씌움 -> 외부 승인 -> `박은 보류`
- 판단: 기존 `old-F-21-원고.md`, `old-F-22-원고.md`를 대체하는 canon 블록

### 2. `F-22-원고.md`

- 역할: 병원 밖 인계 개시
- 기능: `외부 7번`, `R-7 / 외부이송`, `보호자 보류`, `동현`
- 판단: 기존 `old-F-23-원고.md`를 재배치한 canon 블록

### 3. `F-23-원고.md`

- 역할: 바깥 운영선의 생활형 얼굴과 시간표 확인
- 기능: `동현` 호출명 확인 -> 강변 창고 -> `동현 2차` -> `04:40 / 2` -> `04:28 / 2`
- 판단: 기존 `old-F-24-원고.md`, `old-F-25-원고.md`를 대체하는 canon 블록

### 4. `F-24-원고.md`

- 역할: 첫 반격과 운영망의 반사신경 확인
- 기능: 교란 -> `세탁 / 우선` 전환 -> 물류 위장 방식 노출
- 판단: 기존 `old-F-26-원고.md`를 재배치한 canon 블록

### 5. `F-25-원고.md`

- 역할: 세탁 위장 다음 단계 종결
- 기능: 린넨 위장 -> 오염 분리 -> `오염 3번` -> `05:05 전 넘김`
- 판단: 기존 `old-F-27-원고.md`, `old-F-28-원고.md`를 대체하는 canon 블록

## 새 읽기 리듬

재정비 후 이 구간의 단계는 이렇게 읽힌다.

### 블록 1

- 사람이 되밀리고, 이름이 덧씌워지며, 박보다 위의 재분류 권한이 열린다

### 블록 2

- 외부 7번이 실제 병원 밖 인계선으로 작동한다

### 블록 3

- `동현`과 강변 창고 라인이 생활형 운영망으로 드러나고, 출발 시간표가 살아 있는 시스템임이 드러난다

### 블록 4

- 주인공 개입에 따라 운영망이 `세탁 / 우선`으로 형식을 바꾸며 반응한다

### 블록 5

- 세탁물 위장 다음에 `오염 3번 / 05:05`라는 더 깊은 익명화 규칙이 드러난다

## 다음 단계 원칙

이 재정비 뒤 다음 화는 더 이상 `새 코드 발견` 화가 되면 안 된다.

반드시 다음 중 하나를 포함해야 한다.

- 첫 직접 가로채기 시도
- 하윤과 실제 접촉 후 구조 실패
- 팀의 외부망 식별/노출
- 상위 운영자 한 층의 더 직접적인 노출

즉, 29화는 `규칙 추가`가 아니라 `사건 확대` 화여야 한다.

## 최종 판단

이 재배열의 장점은 분명하다.

- 같은 엔진 반복을 줄인다
- `병원 내부 -> 외부 인계 -> 강변 창고 -> 세탁 우회 -> 오염 분리`의 5단 상승선이 선명해진다
- 남는 분량을 이후 실제 충돌과 대가에 쓸 수 있다

한 줄로 정리하면:

21~28 구간의 현재 최종 사용 흐름은 `5블록 체제`이며,
핵심은 `새 코드 하나 더`가 아니라 `판이 한 번씩 바뀌는 5단 전진`으로 읽히게 만드는 것이다.
