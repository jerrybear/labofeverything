# 일본어 공부법 조사

## 이 폴더는 무엇을 담고 있나

이 폴더는 `일본어를 어떻게 공부하면 효율적인가`를 빠르게 파악할 수 있게 정리한 조사 노트다. 비전문가도 바로 적용할 수 있게 `학습 원칙`, `추천 순서`, `주간 루틴`, `주의할 실수`, `출처` 중심으로 묶었다.

## 추천 읽기 순서

### 1. 먼저 큰 방향을 잡고 싶다면

- `01_overview.md`
- 목적: 일본어 공부를 어떤 순서와 원칙으로 해야 하는지 한 번에 이해
- 다루는 내용: 핵심 원칙, 권장 학습 순서, 주간 루틴, 흔한 실수

### 2. 근거를 확인하고 싶다면

- `10_sources.md`
- 목적: 이번 정리에 참고한 자료와 근거 강도 확인
- 다루는 내용: 일본어 학습 가이드, 다독 자료, 간격 반복 관련 연구

## 빠른 한 줄 결론

가장 현실적인 일본어 공부법은 `문자 익히기 -> 기초 문법과 핵심 어휘 -> 간격 반복 복습 -> 쉬운 듣기/읽기 입력 확대 -> 점진적 말하기/쓰기` 순서로, 매일 적게라도 꾸준히 반복하는 방식이다.
