# 아들 양육 주간 체크시트 - 유아기

## 이번 주 핵심

- 목표는 완벽한 통제가 아니라 **안정감, 감정 언어, 안전한 신체 놀이, 예측 가능한 루틴**이다.

## 매일 점검

- [ ] 하루에 한 번 이상 아이 감정에 이름을 붙여줬다.
- [ ] 울음이나 짜증 뒤에서 피곤함, 배고픔, 전환 스트레스를 먼저 봤다.
- [ ] 10분 이상 아이 주도 놀이를 따라가며 반응했다.
- [ ] 큰 소리나 모욕 없이 기준을 말했다.
- [ ] 잠들기 전 루틴을 지키려고 했다.

## 이번 주 점검

### 관계

- [ ] 안아주기, 눈 맞춤, 말 걸기 같은 연결 시간이 충분했다.
- [ ] "남자애가 왜 울어" 같은 말을 하지 않았다.
- [ ] 아이가 도움을 구할 때 비교적 일관되게 반응했다.

### 규칙과 훈육

- [ ] 규칙을 짧게 반복했다.
- [ ] 잘못했을 때 혼내기만 하지 않고 대안을 알려줬다.
- [ ] 체벌, 비꼼, 공개망신 없이 대응했다.

### 놀이와 몸

- [ ] 뛰기, 오르기, 몸놀이 시간을 충분히 줬다.
- [ ] 거친 놀이를 무조건 금지하지 않고 안전 규칙 안에서 다뤘다.
- [ ] 과한 스크린으로 놀이가 밀려나지 않게 했다.

### 부모 점검

- [ ] 아이 행동을 버릇 문제로만 보지 않았다.
- [ ] 내 피로와 짜증이 아이에게 과하게 쏟아지지 않게 조절했다.
- [ ] 배우자 또는 다른 양육자와 기준을 너무 다르게 쓰지 않으려 했다.
