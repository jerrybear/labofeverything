- 목표는 **관계 유지, 존중 기반 규칙, 디지털 점검, 정서 신호 파악, 독립 준비**다.

## 매일 점검

- [ ] 잔소리만 하지 않고 상태를 묻는 대화를 시도했다.
- [ ] 명령보다 질문을 조금 더 사용했다.
- [ ] 스크린 사용이 잠, 기분, 할 일에 어떤 영향을 주는지 봤다.
- [ ] 비난보다 책임 회복에 초점을 맞췄다.
- [ ] 짧게라도 연결되는 순간을 만들었다.

## 이번 주 점검

### 관계와 정서

- [ ] 무뚝뚝함 뒤의 스트레스, 수치심, 외로움을 살펴봤다.
- [ ] 아이를 어린애처럼만 대하지 않았다.
- [ ] 부모가 잘못했을 때 사과하거나 다시 말했다.

### 규칙과 자율성

- [ ] 귀가, 공부, 스마트폰 규칙을 설명 가능하게 유지했다.
- [ ] 협상 가능한 것과 불가한 것을 구분했다.
- [ ] 시간, 돈, 약속 관리 중 하나 이상을 실제로 맡겼다.

### 온라인과 가치관

- [ ] 유튜브, 게임, 쇼츠, SNS 내용을 한 번 이상 대화했다.
- [ ] 여성혐오, 폭력, 과장된 남성성 콘텐츠를 그냥 넘기지 않았다.
- [ ] 성, 동의, 존중, 관계 기준을 피하지 않고 말했다.

### 위험 신호

- [ ] 수면 붕괴, 방 안 고립, 친구 단절, 성적 급락을 체크했다.
- [ ] 자기비하나 공격성 증가가 이어지는지 살폈다.
- [ ] 필요 시 학교·상담·의료 도움을 연결할 준비를 했다.
