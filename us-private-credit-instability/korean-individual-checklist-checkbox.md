# 한국 개인투자자 체크박스 문서

작성일: 2026-03-13

아래는 미국 private credit 관련 상품이나 기사 때문에 불안할 때, 실제로 확인해야 할 항목을 체크박스 형태로 정리한 문서다.

## A. 내가 가진 상품이 뭔지 확인

- [ ] 상품명과 운용사명을 정확히 알고 있다
- [ ] `직접 private credit 펀드`인지 `재간접 상품`인지 알고 있다
- [ ] `비상장 BDC`, `interval fund`, `사모재간접`, `랩/신탁` 중 어디에 해당하는지 알고 있다
- [ ] 기초자산이 미국 middle-market 기업대출이라는 점을 이해하고 있다

## B. 유동성 약관 확인

- [ ] 환매가 `언제든 가능한 구조`가 아니라는 점을 확인했다
- [ ] 환매 가능 주기가 `월/분기/반기` 중 무엇인지 확인했다
- [ ] 약관에 `repurchase`, `proration`, `cap`, `board discretion`, `suspend` 문구가 있는지 확인했다
- [ ] 환매 신청이 초과되면 다음 분기로 자동 이월되는지 확인했다
- [ ] 판매 당시 설명과 실제 약관이 같은지 비교했다

## C. 수익률의 질 확인

- [ ] 분배금이 실제 현금이자 기반인지 확인했다
- [ ] PIK, 평가이익, 일회성 fee에 기대는 구조인지 확인했다
- [ ] 최근 운용보고서에서 `non-accrual`, `amend-and-extend`, `PIK` 언급이 늘었는지 확인했다
- [ ] 높은 배당률이 곧 높은 안전성을 뜻하지 않는다는 점을 이해한다

## D. 가격과 평가 확인

- [ ] 기준가가 일일 시장가격이 아니라 `월말/분기말 NAV`일 수 있다는 점을 이해한다
- [ ] NAV가 안정적이어도 실제 유동성과 처분가는 다를 수 있다는 점을 이해한다
- [ ] 운용사가 valuation, third-party pricing, fair value 같은 표현을 어떻게 쓰는지 확인했다

## E. 한국 투자자라서 추가로 봐야 할 것

- [ ] 환헤지를 하는 상품인지 확인했다
- [ ] 환헤지 비용이 기대수익을 얼마나 깎을 수 있는지 설명받았다
- [ ] 해외 하위 펀드가 여러 개인 재간접 구조인지 확인했다
- [ ] 달러 강세와 환매지연이 동시에 오면 체감 손실이 커질 수 있음을 이해한다

## F. 기사 읽을 때 오해하지 말 것

- [ ] `환매 제한`과 `환매 중단`은 같은 말이 아니라는 점을 구분한다
- [ ] 운용사 이름만 보고 하나의 펀드 사건을 회사 전체 문제로 일반화하지 않는다
- [ ] 숫자 하나보다 `여러 상품에서 같은 구조적 신호가 반복되는지`를 본다
- [ ] `2008년급 위기`라는 표현은 자극적일 수 있으며, 실제론 구조 비교가 필요하다는 점을 안다

## G. 지금 당장 확인하면 좋은 문서

- [ ] 상품설명서 또는 투자제안서
- [ ] 최근 운용보고서 또는 shareholder letter
- [ ] 환매/상환 관련 공지
- [ ] 판매사 설명자료와 실제 약관 비교표

## H. 빨간불 신호

- [ ] 최근 공지에 `oversubscribed`, `proration`, `repurchase limit`, `gate`가 등장했다
- [ ] 분배는 유지되는데 시장 불안 기사와 환매 기사가 동시에 늘고 있다
- [ ] 소프트웨어, 헬스케어 서비스, sponsor-backed 기업대출 익스포저가 과도하다
- [ ] 환헤지 비용 설명이 불분명하거나 최근 갑자기 악화됐다
- [ ] 판매사가 유동성 설명을 지나치게 단순하게 한다

## 마지막 질문

- [ ] 이 상품을 나는 `채권형`으로 착각하고 있지 않은가
- [ ] 내가 감당 가능한 유동성 제약 수준을 넘는 상품은 아닌가
- [ ] 지금 필요한 건 공포 매도가 아니라 `약관과 구조 재확인`이라는 점을 이해한다
