# People Sources

## 사용 기준

- 각 인물마다 최소 3갈래 자료를 모은다: 입문 개설서, 작품 접근 자료, 보조 해설 또는 강의
- 지금 단계에서는 세부 서지보다 자료 유형과 탐색 방향을 정리한다.
