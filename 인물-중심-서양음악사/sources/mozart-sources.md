# Mozart Sources

## 입문 자료

- H. C. Robbins Landon 또는 Jan Swafford 계열의 모차르트 전기: 생애와 빈 활동기 전체 개관
- 모차르트 서간집 발췌본: 경제 문제, 공연 활동, 인간관계를 직접 확인할 때 사용
- 18세기 빈 음악 문화 개설서: 궁정과 시장의 이중 구조를 이해할 때 유용

## 작품 접근 자료

- Bärenreiter 계열의 `Le nozze di Figaro` 보컬 스코어와 대본 번역본
- 후기 교향곡 미니어처 스코어 또는 총보
- Henle 계열의 피아노 협주곡 악보와 해설서

## 음원, 영상, 강의 타깃

- `Le nozze di Figaro` 전막 영상 1편
- 39, 40, 41번 교향곡 비교 청취
- 빈 음악 문화와 다 폰테 오페라를 다루는 입문 강의

## 듣기와 비교 포인트

- 오페라: 인물 관계와 앙상블 장면의 전개를 중심으로 듣기
- 교향곡: 대위법적 밀도와 고전주의 형식의 유연성 확인
- 비교 인물: `../people/haydn.md`, `../people/beethoven.md`
