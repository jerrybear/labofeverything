# Sources

## 목적

- `bibliography.md`를 전체 안내판으로 두고, 이 폴더는 실제 참고자료를 더 세분화해 쌓는 작업 공간으로 사용한다.
- 인물별, 장르별, 도시별 자료를 분리해서 나중에 링크, 판본, 강의, 영상까지 붙일 수 있게 한다.

## 현재 문서

- `people-sources.md`
- `topic-sources.md`
- `source-note-template.md`

## 사용 방식

1. 새 책이나 강의를 찾으면 먼저 해당 문서의 항목 아래에 추가한다.
2. 자료가 많아지면 특정 인물이나 도시를 별도 파일로 분리한다.
3. 가능하면 자료 성격을 함께 적는다: 전기, 개설서, 악보, 영상, 강의, 논문
