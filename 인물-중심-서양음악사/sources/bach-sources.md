# Bach Sources

## 입문 자료

- Christoph Wolff 계열의 바흐 전기: 생애와 도시 이동을 전체적으로 파악할 때 사용
- 루터교 음악 문화 개설서 또는 교회음악사 입문서: 바흐를 예배 실천 안에서 읽을 때 유용
- 멘델스존 이후 바흐 재발견사를 다루는 해설서: 19세기 이후 위상 변화를 볼 때 필요

## 작품 접근 자료

- Henle 계열의 `Das Wohltemperierte Klavier` 우르텍스트 악보
- Bärenreiter 계열의 `Matthaus-Passion` 보컬 스코어와 번역 대본
- 주요 칸타타 선집 해설 자료 또는 BWV 안내서

## 음원, 영상, 강의 타깃

- `Das Wohltemperierte Klavier` 전곡 중 1권 선별 청취
- `Matthaus-Passion` 전곡 실황 영상 또는 음원 1편
- 바흐와 루터교 예배 음악을 다루는 공개 강의 또는 대학 강의 영상

## 듣기와 비교 포인트

- 건반곡: 프렐류드와 푸가의 구조 비교
- 수난곡: 복음사가, 코랄, 합창의 역할 분리해서 듣기
- 비교 인물: `../people/handel.md`, `../people/beethoven.md`
