# People Index

## 사용 기준

- 시대보다 인물의 역할과 영향 관계를 기준으로 정리한다.
- 꼭 작곡가만이 아니라 연주자, 이론가, 교육자, 후원자도 포함한다.
- 각 인물은 개별 문서로 확장할 수 있도록 출발점만 간단히 적는다.

## 핵심 인물군을 보는 방식

- 아래 목록은 연대순이 아니라 영향망과 역할군을 기준으로 묶었다.
- 한 인물은 여러 묶음에 걸칠 수 있지만, 시작점으로 가장 유용한 연결망에 우선 배치했다.
- 개별 문서가 이미 있는 인물은 바로 들어갈 수 있도록 링크를 붙였다.
