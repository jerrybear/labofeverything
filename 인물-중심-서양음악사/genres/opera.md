# Opera Network

## 왜 중요한가

- 오페라는 단순 장르가 아니라 궁정, 도시, 극장, 출판, 스타 시스템이 엮이는 인물 네트워크다.
- 인물 중심 음악사에서는 작품보다 누가 어떤 제도와 무대를 통해 오페라를 바꿨는지 보는 것이 핵심이다.

## 핵심 인물선

- 몬테베르디 -> 륄리 -> 헨델 -> 모차르트 -> 베르디 -> 바그너

## 중심 질문

- 오페라는 궁정 예술에서 공공 극장 문화로 어떻게 이동했는가?
- 작곡가, 가수, 후원자, 극장 운영자는 어떤 방식으로 권력을 나눴는가?
- 같은 장르라도 프랑스, 이탈리아, 독일어권에서 무엇이 달라졌는가?
