# 인물 중심의 서양음악사

## 목적

- 시대 구분보다 작곡가, 연주자, 이론가, 후원자 같은 인물을 중심으로 서양음악사의 흐름을 정리한다.
- 각 인물이 누구에게 영향을 받았고 누구에게 영향을 주었는지 연결 구조를 함께 기록한다.
- 작품, 사조, 제도 변화는 인물 노드와의 관계 안에서 설명한다.

## 시작 질문

- 이 인물은 어떤 음악적 문제를 새롭게 다루었는가?
- 이 인물의 활동 기반은 어디였고 어떤 네트워크 안에 있었는가?
- 대표 작품은 무엇이며 동시대와 후대에 어떤 영향을 주었는가?
- 이 인물을 통해 이전 세대와 다음 세대를 어떻게 이어서 볼 수 있는가?

## 기본 분류

### 인물 유형

- 작곡가
- 연주자
- 지휘자
- 이론가
- 교육자
- 후원자

### 기록 항목

- 생애 요약
- 핵심 활동 시기
- 주요 관계 인물
- 대표 작품 또는 저술
- 음악사적 전환점
- 후대 영향

## 추천 작업 순서

1. 기준 인물군을 먼저 정한다.
2. 각 인물별 개별 노트를 만든다.
3. 영향 관계를 교차 링크로 연결한다.
4. 필요하면 특정 장르나 지역 기준의 보조 노트를 추가한다.

## 현재 작업 문서

- `people-index.md`: 핵심 인물군을 고르고 개별 문서로 확장하기 위한 출발점
- `person-template.md`: 인물별 노트를 같은 형식으로 정리하기 위한 템플릿
- `influence-map.md`: 인물 간 영향, 전승, 후원, 재발견 관계를 연결하는 지도
- `timeline-by-person.md`: 시간 흐름을 유지하되 설명 중심을 인물에 두는 타임라인
- `reading-path.md`: 관심사별로 문서를 읽어 들어가는 추천 경로
- `bibliography.md`: 인물, 장르, 도시별로 바로 참고할 수 있는 기본 자료 선반
- `link-log.md`: 음반, 악보, 강의, 영상 링크 후보를 모아두는 중앙 로그
- `relationship-diagram.md`: 영향망을 시각화 도구로 옮기기 쉬운 다이어그램 초안
- `relationship-diagram-cities.md`: 도시 허브 중심으로 다시 그린 관계 다이어그램
- `relationship-diagram-genres.md`: 장르 중심으로 다시 그린 관계 다이어그램
