# Link Log

## 목적

- 아직 URL을 직접 확정하지 않은 상태에서, 어떤 종류의 링크를 어디에 붙일지 후보를 체계적으로 모은다.
- 나중에 실제 웹 링크를 찾으면 이 문서를 출발점으로 각 인물, 장르, 도시, 소스 문서에 반영한다.

- 강의: 18세기 빈 음악문화, 다 폰테 오페라

- 강의: 영웅성 신화, 후기 양식, 빈 후원 구조

### 드뷔시

### 바그너

## 작업 규칙

1. 실제 URL을 찾으면 먼저 여기 후보 항목을 구체화한다.
2. 검증한 링크만 개별 `people/*.md` 또는 `sources/*.md`에 옮긴다.
3. 링크를 붙일 때는 자료 유형을 함께 적는다: 음반, 악보, 강의, 영상, 논문
