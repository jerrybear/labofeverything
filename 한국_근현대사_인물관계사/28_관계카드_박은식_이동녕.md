# 관계카드: 박은식 - 이동녕

## 1. 관계 기본 정보

- 관계 당사자 A: 박은식
- 관계 당사자 B: 이동녕
- 관계 단위: 인물 대 인물
- 주요 시기: 1919-1925를 중심으로, 임정 제도 전환 국면 포함
- 주요 활동 지역: 상해를 중심으로 한 중국 관내 지역

## 2. 이 관계를 왜 조사하는가

- 이 관계는 임시정부 통합 지원, 위기 수습, 대통령제 폐지와 국무령제 도입으로 이어지는 제도 전환을 읽는 핵심 축이다.
- 박은식은 과도기 지도 구조의 인물이고, 이동녕은 장기적 유지파 지도 축으로 정리되어 있어 두 사람의 연결은 임정 수습 과정을 보여 준다.
- 김구, 이승만, 안창호를 중심으로 보이는 갈등 구조 뒤에서 실제 제도 정비가 어떻게 이어졌는지 파악하는 데 중요하다.

## 3. 핵심 질문

- 박은식과 이동녕은 임정 위기 국면에서 어떤 방식으로 연결되었는가
- 두 사람의 관계는 이승만 면직 이후 과도기 수습과 국무령제 전환에서 어떤 의미를 가졌는가
- 이 관계는 임정 유지 전략과 제도 개편의 결합을 어떻게 보여 주는가

## 4. 관계의 주요 유형

- 주요 유형: 제도·권력 관계
- 보조 유형: 조직 관계, 협력 관계, 갈등 조정 관계
- 자료 수준: 직접 확인과 간접 확인이 함께 존재함

## 5. 확인된 사실

- 박은식은 1919년 상해로 이동해 임시정부 통합 운동을 지원하였다.
- 박은식은 1925년 이승만 면직 이후 제2대 임시대통령으로 선출되었다.
- 박은식은 대통령제의 폐단을 줄이기 위해 국무령제로의 개헌 뒤 사임하였다.
- 이동녕은 1919년 임시의정원 초대 의장으로 선출되었고, 1924년 국무총리, 1926년 국무령을 역임하였다.
- 이동녕 인물카드 기준으로 박은식의 과도기 지도체제와 이동녕의 후속 제도 운영은 같은 수습 국면 안에서 연결된다.
