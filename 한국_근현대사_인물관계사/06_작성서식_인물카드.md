# 한국 근현대사 인물카드 작성 서식

## 이 문서의 목적

- 개별 인물 문서를 같은 구조로 작성할 수 있도록 공통 템플릿을 제공한다.
- 사실 확인, 관계 정리, 해석 쟁점, 출처 표기를 한 문서 안에서 분리해 기록하도록 돕는다.
- 이후 누적될 인물카드의 비교 가능성과 검증 가능성을 높인다.

## 사용 원칙

- 이 서식은 `02_인물선정기준.md`, `03_관계유형분류.md`, `04_출처정리와_검증기준.md`를 실제 문서에 적용하는 틀이다.
- 한 인물의 전 생애를 빠짐없이 서술하려 하기보다, 관계망이 가장 잘 드러나는 핵심 국면을 우선 정리한다.
- 사실과 해석을 분리해 적고, 확인되지 않은 내용은 `유보` 또는 `추가 검증 필요`로 남긴다.
- 문장보다 출처가 먼저라는 원칙을 유지한다.

## 파일명 권장 형식

- `06_인물카드_이름.md`
- 실제 작성이 누적되면 시기 또는 묶음 번호를 앞에 더 붙여 운영할 수 있다.

## 항목별 작성 안내

### 1. 기본 정보

- 생몰년, 활동 시기, 주요 소속은 가능하면 S등급 또는 A등급으로 확인한다.
- 이명, 호, 한자 표기는 검색과 자료 추적에 도움이 될 때만 넣는다.

### 2. 이 인물을 왜 조사하는가

- "유명하기 때문"이 아니라 시기 대표성, 관계망 중심성, 조직 연결성, 매개 역할을 중심으로 쓴다.

### 3. 핵심 질문

- 질문은 단순 전기 요약이 아니라 관계와 변화 중심으로 설정한다.
- 예: 누구와의 관계에서 이 인물의 역사적 의미가 커지는가
- 예: 어느 시점에 협력 관계가 갈등 관계로 전환되는가

### 4. 확인된 사실

- 해석을 섞지 않고 자료로 확인 가능한 문장만 적는다.
- 한 문장마다 출처가 어디에 있는지 추적 가능해야 한다.

### 5. 시기별 정리

- `01_시기구분과_핵심질문.md`의 구간에 맞춰 시기를 나눈다.
- 각 시기마다 활동, 연결 인물, 조직, 사건을 나눠 적으면 이후 관계 카드 작성이 쉬워진다.

### 6. 주요 관계

- 관계는 `03_관계유형분류.md`의 기준에 따라 `주요 유형`과 `보조 유형`으로 나눈다.
- 같은 인물과의 관계도 시기에 따라 바뀌면 별도로 적는다.

### 7. 관계망에서의 위치

- 이 인물이 중심 인물인지, 중개자인지, 특정 갈등 축의 한 축인지 정리한다.
- 단, 이 항목도 반드시 근거를 전제로 써야 한다.

### 8. 해석 쟁점

- 학술적으로 견해가 갈리는 지점만 적는다.
- 가치판단을 먼저 쓰지 말고, 어떤 쟁점에서 어떤 해석 차이가 있는지 적는다.

### 9. 자료 수준 평가

- 이 인물에 대한 S/A/B/C 자료의 밀도를 간단히 적는다.
- 특정 인물이 대중적으로 유명해도 S등급 자료가 약하면 그 점을 명시한다.

### 10. 유보 또는 추가 검증 필요

- 전승, 회고담, 후대 기념 서사처럼 아직 확정할 수 없는 항목을 적는 칸이다.
- 비워두지 말고 조사 공백을 명시하는 용도로 쓴다.

### 11. 출처

- `04_출처정리와_검증기준.md`의 형식을 따른다.
- 가능하면 S등급과 A등급을 먼저 적고, B와 C는 뒤에 둔다.

### 12. 다음에 이어서 볼 인물

- 이 인물과 직접 연결되는 사람을 적는다.
- 이후 조사 확장 경로를 남기는 칸이므로 매우 중요하다.

## 짧은 작성 예시 형식

```md
## 4. 확인된 사실

- [사실] 19XX년 [조직명]에 참여한 기록이 확인된다.
- [사실] 19XX년 [인물명]과 공동 성명에 이름이 확인된다.

## 6. 주요 관계

### 관계 1. [인물명]

- 주요 유형: 협력 관계
- 보조 유형: 조직 관계
- 관계의 성격: [조직명] 활동을 통해 일정 기간 협력
- 확인 근거: 공동 성명, 당시 기사, 학술 논문
- 변화 계기: 19XX년 [사건명] 이후 갈등 발생
```

## 작성 시 피해야 할 것

- 근거 없이 "대표적", "중심적", "가장 영향력 있는" 같은 표현을 쓰는 것
- 하나의 회고 자료만으로 관계를 확정하는 것
- 사실과 해석을 한 문장에 섞는 것
- 현재의 평가를 과거 인물의 동기처럼 서술하는 것
- 같은 조직 소속을 자동으로 친밀 관계로 처리하는 것

## 이 서식의 장점

- 여러 인물 문서를 같은 기준으로 비교할 수 있다.
- 관계 카드와 시기 흐름 문서로 자연스럽게 연결된다.
- 조사 공백과 검증 완료 영역을 함께 보이게 할 수 있다.
