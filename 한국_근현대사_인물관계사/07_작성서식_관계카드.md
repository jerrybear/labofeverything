# 한국 근현대사 관계카드 작성 서식

## 이 문서의 목적

- 인물과 인물, 인물과 조직, 조직과 조직 사이의 관계를 같은 형식으로 기록할 수 있도록 공통 템플릿을 제공한다.
- 관계를 단일 평가가 아니라 형성, 변화, 단절의 과정으로 기록하도록 돕는다.
- 이후 누적될 관계카드의 비교 가능성과 검증 가능성을 높인다.

## 사용 원칙

- 이 서식은 `03_관계유형분류.md`와 `04_출처정리와_검증기준.md`를 실제 문서에 적용하는 틀이다.
- 관계는 감정 표현보다 확인 가능한 행위, 사건, 문서, 조직, 이동 경로를 중심으로 적는다.
- 한 관계는 하나의 유형으로만 고정하지 않고 `주요 유형`과 `보조 유형`을 함께 적는다.
- 사실, 해석, 유보를 분리해서 적는다.

## 파일명 권장 형식

- `07_관계카드_이름A_이름B.md`
- 인물과 조직의 관계일 경우 `07_관계카드_이름_조직명.md`
- 조직과 조직의 관계일 경우 `07_관계카드_조직A_조직B.md`

## 항목별 작성 안내

### 1. 관계 기본 정보

- 관계 단위를 먼저 정해야 문서 범위가 흔들리지 않는다.
- 인물 대 인물 관계인지, 인물 대 조직 관계인지에 따라 확인해야 할 자료가 달라질 수 있다.

### 2. 이 관계를 왜 조사하는가

- "유명한 관계"이기 때문이 아니라, 시기 전환, 조직 재편, 갈등 축, 지역 연결을 설명하는 데 중요하기 때문이라는 식으로 적는다.

### 3. 핵심 질문

- 질문은 관계의 변화와 구조에 집중한다.
- 예: 이 관계는 어떤 계기에서 형성되었는가
- 예: 어느 시점에 협력에서 갈등으로 바뀌었는가
- 예: 이 관계는 더 넓은 조직 재편과 어떻게 연결되는가

### 4. 관계의 주요 유형

- `03_관계유형분류.md`의 분류를 그대로 적용한다.
- 한 관계가 복합적이면 `주요 유형`은 가장 중심적인 것 하나, `보조 유형`은 보완적 요소를 적는다.
- 자료 수준은 `직접 확인`, `간접 확인`, `유보`로 구분한다.

### 5. 확인된 사실

- 확인된 사실은 문서, 기사, 판결문, 선언문, 조직 문건 등에서 직접 추적 가능한 내용만 적는다.
- 해석과 추정은 이 항목에 넣지 않는다.

### 6. 관계 형성 배경

- 관계가 처음 나타난 시점과 계기를 적는다.
- 같은 조직 가입, 공동 선언, 지역 이동, 소개자 등장, 공동 사건 참여 등이 형성 계기가 될 수 있다.

### 7. 시기별 관계 변화

- 관계는 고정값이 아니라 변동 과정으로 기록한다.
- 각 시기마다 `관계 상태`, `확인된 사건`, `변화 요인`을 나눠 적으면 흐름이 선명해진다.

### 8. 관계 유형별 분석

- 모든 항목을 반드시 채울 필요는 없지만, 관계를 설명하는 핵심 유형은 빠뜨리지 않는다.
- 협력과 갈등이 동시에 존재할 수 있음을 전제로 쓴다.

### 9. 관계망에서의 의미

- 이 관계 하나가 어떤 더 큰 구조를 보여 주는지 적는다.
- 예를 들어 정당 재편, 독립운동 노선 분화, 군부 권력 구조, 민주화 네트워크 확대 같은 더 큰 흐름과 연결한다.

### 10. 해석 쟁점

- 학술적으로 견해가 갈리는 지점만 적는다.
- 누가 옳다고 단정하기보다 어떤 쟁점이 왜 갈리는지 적는다.

### 11. 자료 수준 평가

- 이 관계를 뒷받침하는 S/A/B/C 자료의 밀도와 한계를 함께 적는다.
- 특히 관계는 인물 전기보다 자료가 더 파편적일 수 있으므로 공백 표시가 중요하다.

### 12. 유보 또는 추가 검증 필요

- 회고담, 후대 기념 서사, 단일 기사에만 나오는 관계는 유보 항목에 먼저 적는다.
- 유보는 미완성이 아니라 검증 태도의 표시다.

### 13. 출처

- `04_출처정리와_검증기준.md`의 형식을 따른다.
- 가능하면 관계를 직접 보여 주는 S등급 자료를 먼저 적는다.

### 14. 다음에 이어서 볼 관계

- 이 관계와 연결되는 주변 관계를 적는다.
- 관계카드는 관계망 확장의 중심 장치이므로 다음 조사 경로를 남기는 것이 중요하다.

## 짧은 작성 예시 형식

```md
## 4. 관계의 주요 유형

- 주요 유형: 협력 관계
- 보조 유형: 조직 관계
- 자료 수준: 직접 확인

## 5. 확인된 사실

- [사실] 19XX년 공동 선언에 두 당사자의 이름이 함께 확인된다.
- [사실] 19XX년 같은 조직 지도부 명단에 함께 등장한다.

## 7. 시기별 관계 변화

### 시기 A

- 관계 상태: 협력
- 확인된 사건: [사건명]
- 변화 요인: 공동 목표와 조직 활동

### 시기 B

- 관계 상태: 갈등
- 확인된 사건: [사건명]
- 변화 요인: 지도권 경쟁과 노선 차이
```

## 작성 시 피해야 할 것

- "친했다", "멀어졌다"처럼 근거 없는 심리 표현만 쓰는 것
- 조직 동시 소속만으로 협력 관계를 확정하는 것
- 후대 회고담을 생전의 직접 증거처럼 다루는 것
- 관계의 변화 과정을 생략하고 한 단어로 고정하는 것
- 정치적 평가를 관계 사실보다 먼저 쓰는 것

## 이 서식의 장점

- 관계를 단순 인물 일화가 아니라 구조와 변화의 과정으로 기록할 수 있다.
- 인물카드와 함께 읽을 때 상호 참조가 쉬워진다.
- 이후 시기 흐름 문서로 확장하기 좋다.
