# 증거금/예수금 처리 방식 조사

> 국내 주식 거래에서 증거금과 예수금이 처리되는 방식에 대한 조사 노트

## 목적

- 증권 계좌의 자금 흐름(예수금, 증거금, 대용금) 이해
- D+2 결제 시스템과 자금 상태 변화 파악
- 주문가능금액 vs 출금가능금액 차이점 명확화
- 미수금/반대매매 등 엣지 케이스 정리

## 핵심 질문

1. **예수금이란?** - 증권사에 예치한 현금의 정확한 정의
2. **증거금이란?** - 주문 시 필요한 담보의 역할
3. **D+2 결제란?** - 매도 후 실제 현금화까지 2영업일 소요 이유
4. **주문가능금액 ≠ 출금가능금액인 이유?**
5. **미수금/반대매매는 언제 발생하는가?**
